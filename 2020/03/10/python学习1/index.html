    python 数据类型转换表

    在Linux/Unix系统中，你可以在脚本顶部添加以下命令让Python脚本可以像SHELL脚本一样可直接执行即可：

    #! /usr/bin/env python

    修改脚本权限，使其具有执行权限

    $ chmod +x hello.py

    执行以下命令

    ./hello.py

    python 3 注释

    python中单行注释以#开头

    # 这是一个注释
print("Hello, World!")

    多行注释用三个单引号或三个双引号括起来

    #!/usr/bin/python3 
'''
这是多行注释，用三个单引号
这是多行注释，用三个单引号 
这是多行注释，用三个单引号
'''
print("Hello, World!")

    如何输出函数的注释
